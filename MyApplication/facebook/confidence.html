Политика конфиденциальности

1. Сбор и использование информации: WallPosting собирает минимальные необходимые данные для обеспечения функциональности публикации постов в сети Facebook. Это включает в себя информацию о вашем аккаунте и разрешения, необходимые для публикации контента.

2. Авторизация через Facebook: Для использования функций приложения, вы должны авторизовать WallPosting через ваш аккаунт Facebook. Приложение получает доступ только к необходимой информации для выполнения операций по публикации.

3. Использование разрешений: WallPosting запрашивает и использует только те разрешения, которые необходимы для функционирования приложения. Мы не собираем и не храним лишние данные без вашего согласия.

4. Публикация контента: Приложение WallPosting предоставляет возможность публикации контента (текстовых сообщений) на вашей стене в сети Facebook. Мы обрабатываем и передаем только ту информацию, которую вы предоставляете для публикации.

5. Безопасность и хранение данных: WallPosting принимает все необходимые меры для обеспечения безопасности ваших данных. Мы не храним ваши учетные данные Facebook, и доступ к ним защищен шифрованием.

6. Доступ к личной информации: WallPosting не продает, не обменивает и не передает вашу личную информацию третьим сторонам. Мы используем ее исключительно для обеспечения функциональности приложения.

7. Обновления политики конфиденциальности: Мы оставляем за собой право вносить изменения в нашу политику конфиденциальности. Об изменениях будет уведомлено наше сообщество.

8. Вопросы и обратная связь: Если у вас есть вопросы или требуется дополнительная информация о нашей политике конфиденциальности, свяжитесь с нами по адресу sunmeatrich@gmail.com.
